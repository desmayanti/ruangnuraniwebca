# Ruang Nurani - Aplikasi Web Pengajuan Keluhan Warga

Ruang Nurani adalah aplikasi web frontend sederhana yang dirancang untuk memfasilitasi warga dalam menyampaikan keluhan (*uneg-uneg*) kepada pemerintah.
